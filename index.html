🧮 Як рахуються KPI і бонуси

1. Відсоток виконання по метриці:

% метрики = (Факт / План) × 100
Для метрик "менше = краще": % = (План / Факт) × 100
Приклад: факт 95, план 100 → 95%

2. Зважений % KPI менеджера:

% KPI = Σ(% метрики × Вага метрики / 100)
Приклад: метрика A 95% (вага 40%) + метрика B 105% (вага 60%)
= (95 × 40 + 105 × 60) / 100 = 101%

3. Розрахунок бонусу (3 моделі):

📌 Лінійна модель:

Бонус = Базовий × (% KPI / 100)
Приклад: 85% KPI → 85% від базового бонусу

📌 Порогова модель:

До 80% → 0 бонусу
Від 80% → Базовий × (% KPI / 100) × Макс. коефіцієнт

📌 Матриця (лесенка):

% KPI Бонус
< 70% 0%
70% - 89% 50% бонусу
90% - 99% 100% бонусу
100%+ 120% бонусу
